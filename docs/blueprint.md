# **App Name**: SugarCheck Pro

## Core Features:

- Dexcom API Integration: Fetch and display real-time blood sugar data from the Dexcom API.
- Manual Blood Sugar Logging: Allow users to manually log blood sugar levels from their glucometer.
- Comparison Tool: Compare CGM (Dexcom) values with manually logged values to identify discrepancies. The comparison tool can alert user if levels are too high or low, and whether to consider further action such as contacting their doctor. The tool will use reasoning to decide if there is a discrepancy.
- CSV Export: Enable users to download their logged blood sugar data as a CSV file.

## Style Guidelines:

- Primary color: Electric blue (#7DF9FF), representing a futuristic, digital aesthetic.
- Background color: Dark grey (#333333), providing a high-contrast backdrop for the interface.
- Accent color: Neon green (#39FF14), for interactive elements and highlights.
- Body text: 'Roboto Mono', monospace font for a technical, data-driven feel.
- Headline font: 'Orbitron', sans-serif with a futuristic, geometric design.
- Use glowing, minimalist icons with sharp edges to represent Dexcom data, manual logs, and export functions.
- Two-page layout with a clear separation between Dexcom data display and manual logging. Implement a grid-based structure with distinct panels for each data type. Intuitive navigation between the pages with a prominent, animated menu.
